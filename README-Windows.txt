Installing Under Windows:

Required Video Codecs:
From the "RequiredWindowsOnly" directory, install:
    K-Lite_Codec_Pack_1488_Basic.exe



Building under windows:

Releasing under windows:
    1. build the release directory
    2. open a new admin cmd.exe prompt and cd to the release directory
    3. windeployqt.exe neuroscope.exe
